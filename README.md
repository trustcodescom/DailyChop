# DailyChop 🍽️

DailyChop is a fully responsive food ordering website built using HTML, CSS, and vanilla JavaScript.  
It allows users to browse meals, add items to a cart, and place orders via WhatsApp.

---

## 🌐 Live Demo
https://trustcodescom.github.io/DailyChop/

---

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Modern food menu interface
- Add to cart functionality
- Dynamic cart management
- Checkout page
- WhatsApp order integration
- Contact form with validation
- About page with company details
- Google Maps integration
- Optimized images (WebP format)
- Fast loading and lightweight structure
